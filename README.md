# 🤖 MockMentor — AI-Powered Mock Interview Platform

MockMentor is an AI-driven platform that helps students prepare for interviews by generating personalized questions, simulating real interview scenarios, and providing intelligent feedback.

---

## 🚀 Features

- 📄 Upload Resume (PDF)
- 🤖 AI-Generated Interview Questions
- 🧠 Technical & HR Interview Practice
- 📊 Intelligent Feedback System
- 🔐 Google Authentication (Firebase)
- 💳 Credit-Based Access System
- 💰 Razorpay Payment Integration
- 🎨 Smooth UI with Framer Motion

---

## 🧠 Problem Statement

Students often lack exposure to real interview environments and do not receive structured feedback, which leads to low confidence and poor performance in actual placements.

---

## 💡 Solution

MockMentor simulates real interview scenarios using AI. It generates personalized questions based on user data and provides actionable feedback to improve performance.
